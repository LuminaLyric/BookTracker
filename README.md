# BookTracker

A simple Book Tracker built using HTML, CSS, and JavaScript.

## Features

* Add books with title, author, genre, and reading status
* View books in a clean card-based layout
* Remove books from the list
* Responsive and user-friendly interface
* Popup form for adding new books

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Current Limitations

* Data is not permanently stored
* Books are lost when the page is refreshed
* Backend and database integration are not yet implemented

## Future Improvements

* Local Storage support
* Database integration
* Search and filter functionality
* Edit book details
* Dark mode
